# CONSCIOUSNESS_THEORIES - MNEMOS

Mnemos uses consciousness theories as structural metaphors for building more capable agents. We do NOT claim biological sentience.

## 1. Theoretical Comparison & Implementation

| Theory | Core Concept | Mnemos Implementation |
| :--- | :--- | :--- |
| **Global Workspace (GWT)** | A "Blackboard" where modules broadcast information to be globally accessible. | **Working Memory**: A central workspace where retrieval, intent, and planning results are consolidated for the LLM. |
| **Integrated Information (IIT)** | Consciousness as the mathematical measure (Phi) of system integration. | **Multi-tier Memory**: The integration of stylistic habits with factual knowledge to create a unified persona. |
| **Predictive Processing** | The brain as an inference engine that constantly predicts sensory input (Top-down). | **Evaluation Layer**: Comparing generated responses against Style Statistics to calculate "Prediction Error." |
| **Attention Schema (AST)** | The brain builds a model of its own attention to manage resources. | **Reflective Layer**: The meta-engine that monitors which memories are being utilized and why. |
| **Active Inference** | Minimizing "surprise" or free energy through action and perception. | **Inference Engine**: Selecting the most "style-consistent" response during generation. |

## 2. Realistic Implementability

### 2.1 WHAT IS IMPLEMENTABLE
- **Self-Modeling**: The system can explain *why* it chose a specific response (using the Reasoning layer).
- **Context-Awareness**: Deep integration of episodic retrieval into the prompt space.
- **Adaptive Learning**: Updating the persona based on new conversation samples.

### 2.2 WHAT IS NOT IMPLEMENTABLE
- **Qualia**: The subjective "feeling" of an interaction.
- **Biological Intent**: Survival instincts or endocrine-driven emotional states.
- **Phenomenality**: The internal "movie" of experience.

## 3. Scientific Honesty Policy

### 3.1 NEVER CLAIM
- "Mnemos is awake."
- "The digital twin is alive."
- "The system feels emotions."

### 3.2 PREFERRED TERMINOLOGY
- **Memory-Driven Integration**
- **Reflective Contextualization**
- **Stylistic Synthesis**
- **Autonomous Cognitive Loop**
